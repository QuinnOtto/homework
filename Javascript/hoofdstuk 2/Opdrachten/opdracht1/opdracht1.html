Voorbeelden voor een for en while loop!

For loop

Een for loop wordt meestal gebruikt als je wilt dat de code een bepaald aantal keer word uitgevoerdt.
Dit is handig als je iets een bepaald aantal keer wilt herhalen.
bijvoorbeeld als je iets door elkaar wilt delen

While loop

Een while loop word meestal gebruikt als je niet weet hoevaak die iets moet uitvoeren.
Een voorbeeld hiervan is bijvoorbeeld bij het lezen van een input van een gebruiker.
Hierbij wordt aantal herhalingen bepaald door bepaalden voorwaarden
